# Phase 1 — Сайт "Бодрость и Сила"

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Собрать работающий Next.js-сайт с тремя направлениями (баня, релаксология, кунгфу), страницами мастеров, локациями и кнопкой записи через YClients.

**Architecture:** Статический контент хранится в TypeScript-объектах (`src/lib/data/`). Страницы генерируются через App Router (`generateStaticParams`). Модель данных нормализована: мастера, направления, услуги и локации — независимые сущности, связанные через id/slug. Деплой — отдельный план.

**Tech Stack:** Next.js 14 (App Router), TypeScript, Tailwind CSS, shadcn/ui, Vitest + @testing-library/react (тесты калькулятора)

```typescript
'use client'

import { useState } from 'react'
import { allAddOns } from '@/lib/data'

interface Props {
  className?: string
}

const BANYA_ADDONS = allAddOns.filter((a) =>
  ['addon-venik', 'addon-scrub', 'addon-aroma', 'addon-sound', 'addon-tea', 'addon-body-work'].includes(a.id)
)

function peopleLabel(n: number) {
  if (n === 1) return '1 человек'
  if (n >= 2 && n <= 4) return `${n} человека`
  return `${n} человек`
}

export function BathCalculator({ className = '' }: Props) {
  const [people, setPeople] = useState(1)
  const [selected, setSelected] = useState<Set<string>>(new Set())

  function toggle(id: string) {
    setSelected((prev) => {
      const next = new Set(prev)
      next.has(id) ? next.delete(id) : next.add(id)
      return next
    })
  }

  const selectedAddOns = BANYA_ADDONS.filter((a) => selected.has(a.id))

  return (
    <section className={`rounded-2xl border border-stone-200 p-6 ${className}`}>
      <h2 className="text-lg font-medium text-stone-900 mb-6">Собрать программу</h2>

      <div className="mb-6">
        <label htmlFor="people-slider" className="block text-sm text-stone-600 mb-2">
          Количество человек
        </label>
```
